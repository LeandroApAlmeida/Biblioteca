<h3>Sistema para gestão do acervo pessoal</h3>

<br>

Aplicativo Asp.Net para gestão do acervo pessoal. O sistema inclui as seguintes funcionalidades:

<ol>

<li>Cadastro de livros do acervo;</li>

<li>Descarte de livros do acervo;</li>

<li>Doação de livros do acervo;</li>

<li>Empréstimo de livros do acervo;</li>

<li>Controle de acesso via login e senha;</li>

<li>Registro de log de sessão.</li>
  
</ol>
