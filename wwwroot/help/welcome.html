Seja bem-vindo(a)!


Neste tutorial você encontra a ajuda necessária para utilizar o sistema ©Biblioteca, uma aplicação web desenvolvida para facilitar o gerenciamento do seu acervo pessoal. Nela, você tem disponíveis diversas funcionalidades, sendo as principais delas:



Gerenciamento de livros no acervo: permite o cadastro, alteração, exclusão, descarte, doação e empréstimo de livros disponíveis no acervo.



Gerenciamento de livros descartados: permite o cadastro, alteração e exclusão de livros que foram descartados do acervo.



Gerenciamento de livros doados: permite o cadastro, alteração e exclusão de livros que foram doados para terceiros.



Gerenciamento de empréstimos: permite o cadastro, alteração, devolução e exclusão de empréstimos de livros.



Você pode explorar cada uma das funcionalidades clicando nas opções do menu à esquerda desta página. O menu do tutorial está organizado de tal forma que reflete a ordem dos menus da aplicação, a saber:



Login: Não há um menu Login, mas como este é o primeiro processo a ser realizado assim que se acessa a aplicação, então ele aparece como o primeiro item a ser consultado no tutorial.



Menu Livro: O menu Livro leva à página Detalhes do Livro. Nesta página são mostrados todos os dados do livro que está em exibição, como título, subtítulo, autor, editora, ISBN, etc. A página tem também uma barra de navegação que permite navegar de livro em livro, em ordem alfabética dos títulos, por todos os livros cadastrados, esteja o livro disponível no acervo, tenha sido descartado, doado ou esteja emprestado. A barra de navegação disponibiliza também um filtro que permite pesquisar por um livro específico para exibir.



Menu Acervo: O menu Acervo leva à página Livros no Acervo. Esta página permite a manutenção dos cadastros dos livros que se encontram disponíveis no acervo.



Menu Descarte: O menu Descarte leva à página Livros Descartados. Esta página permite a manutenção de livros que foram descartados em razão de danos, perda, ou outro motivo.



Menu Doação: O menu Doação leva à página Livros Doados. Esta página permite a manutenção de livros que foram doados para terceiros.



Menu Empréstimo: O menu Empréstimo leva à página Empréstimos. Esta página permite a manutenção dos empréstimos, que inclui o cadastro da retirada do livro e da sua devolução.



Menu Pessoa: O menu Pessoa leva à página Pessoas. Esta página permite a manutenção de pessoas para quem foram feitas doações de livros e empréstimos.



Menu Usuário: O menu Usuário é exclusivo para o Administrador do sistema e leva à página Usuários. Esta página permite a manutenção das contas de usuários para acesso ao sistema.



Menu Log: O menu Log é exclusivo para o Administrador do sistema e leva à página Log de Sessão. Esta página exibe os dados de todas a sessões iniciadas, assim que um login é realizado.



Menu Opções: O menu Opções leva à pagina de personalização. Esta página permite a um usuário configurar algumas opções de exibição para seu perfil específico, como a cor que vai ser utilizada para destacar os livros descartados, doados e emprestados, se vai ser exibida ou não a legenda de cores no rodapé das páginas e o formato do arquivo de relatório (PDF ou HTML).



Menu Sobre: O menu Sobre leva à página de créditos da versão. Esta página exibe a licença do programa e os créditos da versão, bem como dá acesso ao tutorial.



Atalhos de teclado: O último item de menu do tutorial também não é um menu da aplicação. Trata-se de uma lista dos atalhos de teclado que podem melhorar a usabilidade da aplicação.



Você pode também retornar a esta página de boas vindas clicando no título Biblioteca acima dos item do menu.
